# TabAlign

A Chrome extension that automatically organizes active browser tabs by domain, with a fallback group for single-tab domains.

## Features

- **Domain-Based Grouping**: Instantly groups tabs by domain.
- **Misc Group Fallback**: Collects single-tab domains into a unified "Misc" group to prevent tab clutter.
- **Auto-Collapse**: Collapses all tab groups upon action for a clean tab bar.
- **Auto-Grouping**: Optional toggle to group tabs dynamically as you browse.
- **A/B Theme Toggle**: Hot-swappable visual themes (Vercel Light, Vercel Dark, Neo-Brutalist, and Glass Dark).

## Installation

1. Clone this repository or download the source code.
2. Open Google Chrome and navigate to `chrome://extensions/`.
3. Enable the **Developer mode** toggle in the top-right corner.
4. Click **Load unpacked** in the top-left corner.
5. Select the extension directory.
